A propos de notre application

Définition générale→

ESIST History Manager est un outil de gestion de l’historique des anciens étudiants de l’ESI .

Objectif général→

Automatiser la gestion de l'historique des anciens étudiants de l'ESI en résolvant les problèmes de l'ancienne application déjà utilisée 'DBASE'.

Droits d'usage→

L'application est destinée uniquement au service de scolarité de l’ESI (un seul utilisateur et mono-poste).

Définition des mots de passe→

Les deux profils Administrateur et Employé

Profil d'employé→

En étant connécté en tant qu'un employé, l'application vous autorise de rechercher un étudiant, imprimer ses sorties, effectuer un classement ou des statistiques ainsi qu'imprimer un PV.

Profil d'administrateur→

En tant qu'administrateur, vous êtes autorisé d'effectuer toutes les opérations disponibles à l'employé, avec le privilège de modifier des informations spécifiques d'un étudiant, et de modifier les mots de passe des deux profils.

Changement des mots de passe

Accessibilité→

Cette opération est accessible à l'administrateur uniquement, il peut modifier le mot de passe de son profil, ainsi que celui du profil de l'employé.

Comment changer les mots de passe?→

Vous devez vous connecter d'abord en tant qu'administrateur. Dans la page principale de l'application, vous trouverez un bouton à droite de la page. Cliquez sur ce bouton. Sur la page de la modification des mots de passe, veuillez choisir le mot de passe que vous souhaitez changer, saisissez le nouveau mot de passe, et confirmer le changement.

Problèmes survenant du manque des informations de la base de données

Recherche→

Si vous effectuez une recherche d'un étudiant et vous ne trouvez aucun résultat, il se peut que l'étudiant concerné manque quelques informations générales (Nom, Etat civile, ..).

RN et RNG et Attestation→

Si un étudiant est accessible par la recherche mais une des sorties (Relevé de notes, Relevé de notes global, Attestation) est inaccessible, cet étudiant manque des informations d'inscription (Moyenne, Notes..).

Classement, Statistiques et PV→

Ces opérations sont effectués sur les étudiants qui figurent dans la base de données principale. Leurs résultats ne représentent pas les étudiants qui sont dans la table à champs vides.

Solution des problèmes


Comment faire?

Notre application manipule les enregistrements chargés dans la base de données principale. Vous trouverez tout les enregistrements qui manquent d'informations (s'il y en a) stockés dans la table à champs vides.

Pour résoudre les problèmes mentionnés, il suffit de lancer le rechargement des données à partir de la table à champs vides après le traitement des enregistrements. Vous trouverez le bouton du rechargement dans la page principale de l'application.

Quels sont les fonctionnalités générales?

Les fonctionnalités générales se trouvent dans la page principale. Elles ciblent une promotion, une section, ou un groupe d'étudiants, mais pas un seul.

Recherche→

La recherche vous affiche l'étudiant (ou les étudiants) vérifiant les critères que vous saisissez.

Classement→

Cette fonctionnalité effectue le classement de fin de cursus d'une promotion de votre choix.

Statistiques→

Les statistiques vous permet d'examiner clairement les données de votre base, selon plusieurs critères.

PV→

Avec cette fonctionnalité, vous pouvez avoir le Procés verbal global d'une promotion d'étudiants.

Opérations/Sorties pour un seul étudiant

Modification→

L'application permet à l'administrateur de modifier les informations de l'état civile d'un étudiant à sa demande. Les informations de l'inscription d'un étudiant ne peuvent pas être modifiées.

Impression des sorties→

(Relevé de notes, Relevé de notes global, Attestation)

Chaque étudiant, qui est présent dans la base de données principale de l'application, a la possibilité d'avoir son propre relevé de notes, relevé de notes global, et son attestation.

Détails→

Après la séléction d'un étudiant, vous pouvez vérifier tout ses détails avec cette fonctionnalité, avant d'effectuer une des autres opérations.

Comment sélectionner un étudiant ?

Les étudiants sont accessibles après avoir fait une recherche, ou bien un classement de fin de cursus.

Quand vous sélectionnez un étudiant à partir du résultat (de la recherche ou du classement), vous aurez accés aux opérations en bas de la page. Pour vérifier que vous avec sélectionné le bon étudiant, vous pouvez vérifier ses détails. Ensuite, vous pouvez lui imprimer ses sorties, ou modifier ses informations si vous êtes administrateur.
